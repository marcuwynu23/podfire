PodFire

Self-hosted deployment platform

Deploy from your repos with agent-based builds, one dashboard, automatic routing, and managed DNS.

  • Agent-based
  • One dashboard
  • Auto routing
  • Managed DNS

Features

  • Self-hosted

    Run everything on your own infrastructure. No vendor lock-in—your code, your servers, your control.

  • Agent-based deploys

    Agents connect to the platform, pull your code, build and deploy. Scale by adding more agents when you need more capacity.

  • Repo-based sign-in

    Sign in with your code host. Access tokens stored encrypted. Link repos and branches to create and manage services.

  • Framework detection

    Detects your app type and applies the right build and deploy pipeline automatically—no manual config.

  • Dashboard

    Manage apps, view deployment history and live logs, trigger deploys, and monitor connected agents from one place.

  • Edge gateway

    Deploy and manage the edge gateway from the dashboard. Routes your domains to your apps automatically.

  • Managed DNS

    Automatically create and assign domains for your apps—no manual DNS setup. Point your apps to the world in one place.

  • Auto-deploy

    Optional auto-deploy: the platform checks for new commits and deploys for you so you stay up to date without manual steps.
